## 🚀 How to run

For clone and run this app, you need to have [Git](https://git-scm.com) and [xCode][xcode] installed in your PC.

## 🖥 Prints

<img alt="Swiftinder" src="https://github.com/allexis096/swiftinder/blob/main/tinder/Assets.xcassets/tinder-gif.dataset/tinder.gif" width="200" />

### 🌀 Cloning the repository

```bash
# To clone
$ git clone https://github.com/allexis096/swiftinder.git

# Access the project folder in your terminal
$ cd swiftinder
```

### 🧭 Running the app

```bash

# Running the application
$ Choose Iphone 12 in xCode and click Play to show the Emulator

```
